## ✨ Overview

**Goomi – Minions & Monsters** is a fun front-end project that recreates a cute cartoon character using only **HTML** and **CSS**. It demonstrates how creative illustrations can be built entirely with CSS shapes, positioning, gradients, and animations—without using images.

---

## 🚀 Features

- 🎨 Pure HTML & CSS Artwork
- 👀 Cartoon Character Illustration
- 📮 Stamp-Style Design
- 📱 Responsive Layout
- ⚡ Lightweight & Fast
